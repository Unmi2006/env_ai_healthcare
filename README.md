# ⚡ AI Sustainability Dashboard v3

An advanced **Streamlit-based AI Sustainability Dashboard** that monitors, simulates, and analyzes the environmental impact (CO₂ emissions, power usage) of AI models in real-time.

---

## 🚀 Features

### 🔴 Real-Time Monitoring
- Live simulation of:
  - Queries
  - CO₂ emissions
  - Power consumption
- Dynamic charts with anomaly detection
- Forecasting using Holt’s Linear Smoothing

### 📊 Analytics
- Smoothed trends
- Correlation heatmaps
- CO₂ forecasting
- Efficiency scoring

### 📈 Advanced Insights
- 3D visualization of system behavior
- Load vs Emissions analysis
- Stress gauge
- Anomaly detection

### 🧠 AI Insights (Claude API)
- AI-based sustainability recommendations
- Risk detection (LOW / MEDIUM / HIGH)
- Chat with AI sustainability expert

### 🤖 Model Comparison
- Compare multiple AI models:
  - CO₂ emissions
  - Power usage
  - Efficiency
  - Carbon cost

### 📁 Export Options
- Download CSV data
- Generate PDF sustainability report
